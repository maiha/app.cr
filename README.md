# app.cr [![Build Status](https://travis-ci.org/maiha/app.cr.svg?branch=master)](https://travis-ci.org/maiha/app.cr)

Private composite shards to build application easily.

- crystal: 0.26.1

## Install

```yaml
dependencies:
  app:
    github: maiha/app.cr
    version: 0.5.0
```

## Usage

```crystal
require "app"
```

works as same as

```crystal
require "core-ext"
require "try"
require "opts"
require "shard"
require "toml-config"
```
